
Windows
Download GLFW from http://www.glfw.org and put glfw3.lib library here.
Download GLEW from http://glew.sourceforge.net and put glew32.lib here.

OSX
Download and build OSX GLFW from http://www.glfw.org
Download and build OSX GLEW from http://glew.sourceforge.net
Or use brew to install GLFW and GLEW.
